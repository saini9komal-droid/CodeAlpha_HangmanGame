# CodeAlpha Hangman Game

## Description
This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

## Features
- Random word selection
- 6 incorrect guesses allowed
- Displays guessed letters
- Win/Lose conditions

## Technologies Used
- Python 3

## How to Run
1. Clone the repository
2. Open terminal
3. Run:
   python hangman.py
